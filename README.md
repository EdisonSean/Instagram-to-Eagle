# ins-eagle-sync

Windows CLI tool for downloading Instagram content with `gallery-dl`, staging it locally, and importing the downloaded files into Eagle through the Eagle Local API.

This repository is intentionally scaffolded first. The current code focuses on clear boundaries, testable helpers, and a project layout that can grow into the full sync/import workflow.

## Goals

- Download Instagram content with `gallery-dl`.
- Support two URL modes:
  - Author sync: `https://www.instagram.com/username/`
  - Single post import: `https://www.instagram.com/p/shortcode/` or `https://www.instagram.com/reel/shortcode/`
- Download into a staging directory instead of writing directly into an Eagle library.
- Read metadata JSON generated by `gallery-dl`.
- Import local files into Eagle with the Eagle Local API.
- Store imported `shortcode + media index` entries in `imported.json` to avoid duplicate imports.
- Start with CLI only. No GUI and no Eagle plugin for now.

## Setup

```powershell
py -m venv .venv
.\.venv\Scripts\Activate.ps1
py -m pip install -r requirements.txt
Copy-Item config.example.json config.json
```

Edit `config.json` before running real downloads/imports.

## CLI

Basic command shape:

```powershell
py -m ins_eagle_sync.cli detect "https://www.instagram.com/p/SHORTCODE/"
py -m ins_eagle_sync.cli sync "https://www.instagram.com/username/" --config config.json
py -m ins_eagle_sync.cli import "https://www.instagram.com/p/SHORTCODE/" --config config.json --folder-id YOUR_EAGLE_FOLDER_ID
```

The `detect` command is available now. Download/import commands are intentionally minimal placeholders until the workflow is implemented step by step.

## Metadata and Import Mapping

Planned Eagle fields:

- `name`: first `title_caption_chars` caption characters + shortcode + media index
- `website`: Instagram original post URL
- `annotation`: full caption, author, date, shortcode, source URL
- `tags`: `instagram`, `author:xxx`, `shortcode:xxx`, and hashtags
- `folderId`: target Eagle folder ID

Duplicate prevention key:

```text
shortcode:media_index
```
